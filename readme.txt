We will be adding the Axios package to the React.js Application.
It will help us request data from a development API that we set up with JSON server.
Axios makes requesting data easier than using the fetch API.

Adding the Axios as a new dependency
	npm i axios -S

To lunch JSON server:
	npx json-server -p 3500 -w data/db.json


To clean the application from what we do not use:
    npm uninstall @testing-library/jest-dom
    npm uninstall @testing-library/react
    npm uninstall @testing-library/user-event
    npm uninstall web-vitals
